## Description
CSV test 7.14 — Attempt to withdraw more ETH than the deposited balance.

The application generates the voucher regardless; the L1 revert happens when
the voucher is executed against the EtherPortal contract with insufficient
balance in the application's held funds.

## Steps
1. Send a withdraw payload for an amount exceeding any deposited balance.
2. Verify the voucher is still generated by the application.
3. (Manual) Execute the voucher on-chain and confirm it reverts.

## Expected Behaviour
- Application generates a withdrawEther voucher for the requested amount.
- On-chain execution of the voucher reverts with insufficient funds error.
