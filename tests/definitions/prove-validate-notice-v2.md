## Description
CSV test 7.1 — Submit an input that produces a notice, then prove and validate
the notice on the base layer using the rollups CLI.

## Steps
1. Submit a ping input that produces a notice.
2. Poll until the notice appears in cartesi_listOutputs.
3. Validate the notice Merkle proof on-chain via CLI.

## Expected Behaviour
- Notice output is generated by the application.
- Merkle proof validates successfully against the on-chain contract.
